## Installation

```bash
bun add @fern-ui/color-picker
```

Or copy `color-picker.tsx` and `color.ts` into your project.

## When to use what

- **`onChange`** fires every frame of a drag — use it for anything you are
  painting live.
- **`onChangeComplete`** fires once on settle — use it for saves, undo entries,
  and anything expensive.
- **`format`** picks the model the picker opens in, and the format your
  callbacks receive.
- Turn things **off** to get a smaller picker; everything is on by default.

## Keyboard

Arrow keys step 1, Shift+arrows step 10, Home/End jump — on the field and both
sliders. Channel inputs are spinbuttons that commit on blur or Enter; invalid
input reverts instead of clamping.

Both callbacks receive the full color as a second argument — `hex`, `rgb`,
`hsl`, `hsv` and `alpha`. The conversion helpers are importable on their own:

```tsx
import { formatColor, parseHex, rgbToHsv } from "@fern-ui/color-picker/color"
```
