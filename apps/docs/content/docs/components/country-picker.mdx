Search matches four things at once: the country name, the two-letter code, the
dial code, and common aliases — so `UK`, `GB`, `+44` and `britain` all find the
United Kingdom. Diacritics fold, so `cote` finds Côte d'Ivoire.

## Examples

### Pinning common countries

195 countries in alphabetical order buries the handful most of your traffic
comes from. `priority` lifts them above the list.

<Preview>
  <PreviewDemo>

```tsx
<CountryPicker

  onChange={setCountry}
  priority={["US", "GB", "IN", "DE", "JP"]}
/>
```

### Without dial codes

Turning them off also removes them from the search index, so a query of `44`
matches nothing rather than surprising you with the UK.

### Flags

Flags are served as static SVGs, resolved from the country code. Copy
`public/flags` into your own project, or point `flagSrc` wherever you host
them.

```tsx
<CountryPicker flagSrc={(code) => `https://cdn.example.com/${code}.svg`} />
```

Each is a 512×512 circle-cropped SVG averaging 0.6KB, loaded lazily — only the
rows you scroll to are fetched.

### Keyboard

Arrows move, Home and End jump to the ends, PageUp and PageDown move by ten.
Enter selects, Escape closes, Tab closes and moves on. The trigger opens on
Enter, Space or ArrowDown.
