
## When to use what

- **`label`** names the file. Falls back to `lang` when omitted, so the bar
  always says what you are looking at.
- **`standalone`** rounds all four corners. Leave it off when the block is
  fused to a preview above it.
- **`lineNumbers`** adds the gutter. Off by default — numbers are noise unless
  you are pointing at a specific line.
- Blocks taller than 150px collapse behind an **Expand code** button. Shorter
  ones render whole, with no button.

## Rendering a source file

`ComponentSource` reads a file from a package and pipes it through the same
viewer, so a copy-paste component can show exactly what you would copy.

```tsx
<ComponentSource pkg="color-picker" file="color-picker.tsx" />
```
